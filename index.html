pkg-sbbs

Debian packaging for Synchronet BBS

Tools:
Navigation: Home | pkg-sbbs | sbbsdev | mbsebbs

Synchronet BBS is a BBS system supporting multiple simultaneous users with hierarchical message and file areas, multi-user chat, and BBS door games. A request (#418776) has been submitted for it to be added to the Debian distribution and this project is some work towards that end. See also the sbbsdev project here, which can be used to enable development of it on Debian based systems.

  • The debian-314 branch in the repository is used for the development of the debian packaging directory for the v3.14 versions of Synchronet BBS.
  • The debian-315 branch in the repository is used for the development of the debian packaging directory for the v3.15 versions of Synchronet BBS.
  • The main-314 branch in the repository contains the code for version v3.14 of the Synchronet BBS application, including that imported from the v3.14 distribution archives.
  • The master branch in the repository contains code for the Synchronet BBS application itself, including that imported from the authors CVS Repository and/or distribution archives.
